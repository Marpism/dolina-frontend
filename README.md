# dolina.shop

Репозиторий содержит frontend-часть проекта [dolina.shop] (https://dolina.shop/). 

Dolina.shop это работающий интернет-магазин украшений из натуральных камней.
На нём можно изучить ассортимент, выбрать и заказать доставку товаров в любую точку России. 

## Функционал 

### 1. Каталог 
  - сортировка товаров по цене, новизне и размеру скидки
  - фильтрация товаров по размерам, камням, цветам и другим характеристикам
  - смена фотографий товара со страницы каталога по свайпу на мобильной версии и по наведению курсора на десктопной версии сайта
  - возможность добавить товар в "избранное" 
  - поиск по каталогу

### 2. Карточка товара 
  - выбор размера изделия 
  - добавление в корзину
  - добавление в избранное
  - попап быстрого заказа в один клик
  - вывод развернутой информации о камнях в изделии
  - мини-попапы с дополнительными преимуществами товара 
  - блок рекомендуемых товаров 
  - информация о количестве добавлений товара в избранное
  - цвет кнопки "в корзину" соответствует цвету изделия 

### 3. Корзина 
  - возможность удалить товар из корзины или добавить его в избранное
  - подсчет суммы заказа без учета стоимости доставки
  - попап быстрого заказа в 1 клик 
  - переход к оформлению заказа 

### 4. Оформление заказа 
  - расчет стоимости доставки разными курьерскими службами в зависимости от выбранного населенного пункта
  - прогрессивная скидка на доставку в зависимости от суммы заказа
  - возможность оплатить заказ сразу или наложенным платежом в зависимости от выбранного способа доставки
 

### 5. Личный кабинет пользователя 
  - возможность изменить адрес электронной почты и имя 
  - список заказов со статусом 
  - отслеживание заказов 
  - избранное

### 6. Доставка и оплата 
  - калькулятор доставки разными курьерскими службами в выбранный населённый пункт 


## Технологии 
React SPA, JS, CSS, файловая структура БЭМ



## Разработка

Над проектом работали два человека: фронтенд- и бэкенд-разработчик. 
Бэкенд написан на Node.js с использованием Express.js. 

Первичная разработка велась в сжатые сроки и была завершена за 4 месяца, после чего сайт начал работать и принимать заказы. 


## Планы по доработке проекта 

  - Настроить аутентификацию через ВК;
  - настроить подтверждение регистрации по имейлу;
  - сделать форму обратной связи в "контактах" и на странице оформления заказа;
  - улучшить валидацию на всех формах на фронтенде;
  - сделать более продвинутый кликабельный слайдер с подборками товаров на главной;
  - реализовать навигацию по хлебным крошкам;
  - сделать подсказки в поиске по сайту.
